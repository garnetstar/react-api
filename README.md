Frontend
----
**vývoj**:  
`cd frontend`  
`npm run start`  

 **build pro produkci**  
 `cd frontend`  
 `npm run build` 
 
 **deploy**  
 `bash deploy.sh + jméno větve co se bude nasazovat`  
 pozor, lokální prostředí musí být na stejné větvi co se bude nasazovat aby se frontend vybuyldil ve stejné verzi
